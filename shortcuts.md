
## vscode 

| function           | macOS         | Windows and Linux |
| ------------------ | ------------- | ----------------- |
| copy line          | ⇧ + ⌥ ↓ / ↑   |                   |
| delete line        | ⇧ + ⌘ K       | Ctrl+Shift+K      |
| foldAll            | ⌘ + K, ⌘ + 0  |                   |
| unfold All         | ⌘ + K, ⌘ + J  |                   |
| fold/unfold region | ⌥ + ⌘ + [ / ] |                   |
| multi cursor       | ⌥ + ⌘ ↑/↓     |                   |
| join lines         | ⌃ J           |                   |
| toggle word wrap   | ⌥ Z           |                   |
