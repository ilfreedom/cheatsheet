# editor shorcuts

## vscode

| function           | macOS         | Windows and Linux |
| ------------------ | ------------- | ----------------- |
| Move Line Up/Down  | ⌥ ↓ / ↑       |                   |
| Copy line          | ⇧ + ⌥ ↓ / ↑   |                   |
| Delete line        | ⇧ + ⌘ K       | Ctrl+Shift+K      |
| Join lines         | ⌃ J           |                   |
| New lines          | ⌘/⌘+⇧ + Enter |                   |
| FoldAll            | ⌘ + K, ⌘ + 0  |                   |
| Unfold All         | ⌘ + K, ⌘ + J  |                   |
| Fold/Unfold region | ⌥ + ⌘ + [ / ] |                   |
| Multi cursor       | ⌥ + ⌘ ↑/↓     |                   |
| Toggle word wrap   | ⌥ Z           |                   |

## intellij

| function              | macOS           | Windows and Linux |
| --------------------- | --------------- | ----------------- |
| Move Line Up/Down     | ⌥ ⇧ Up/Down     |                   |
| Delete Line           | ⌘+Backspace     |                   |
| Join line             | ⌃ + ⇧ + j       |                   |
| Expand/collapse all   | ⌘ ⇧ + / ⌘ ⇧ -   |                   |
| Expand/collapse block | ⌘ + / ⌘ -       |                   |
| Navigate bw/fw        | ⌘ + ⌥ ← / ⌘ ⌥ → |                   |
| Navigate last edit    | ⌘ + ⇧ Backspace |                   |
